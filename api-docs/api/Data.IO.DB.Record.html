Class Record

Light-weight base class for all record types. It must not be instantiated directly, instead create a subclass of Record and override SetTuple in the derived class - or use the GenericRecord class that works with any table.

SetValues(IList<Object>)

Takes a list of objects (tuple) holding the values (as object) of the record. IMPORTANT: Must be overriden in subclasses, and each value must be assigned according to the subclass' specification, ie assigned to corresponding field or property, or added to a list. NOTE: do not call the base class implementation, it will throw NotImplementedException.
