Class ScriptBuilder

Create C# scripts reliably using a descriptive API that only leaves you to fill in the custom body lines of methods and other items.

Supports nested types, usings, namespace, methods, constructors, fields, properties, indexers and more in the future. Most C# keywords such as abstract, partial, static are supported as well.
