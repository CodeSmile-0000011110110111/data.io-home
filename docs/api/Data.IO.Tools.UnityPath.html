GetSQLiteDatabaseFilePath(String, PathRoot)

Get the path to a SQLite database files. The extension can be omitted, in this case all variations of common sqlite file extensions (".db", ".sqlite", ".sqlite3" etc) are checked if a file exists. If no file is found with any of the extensions, it will return the file path with default extension ".db".
