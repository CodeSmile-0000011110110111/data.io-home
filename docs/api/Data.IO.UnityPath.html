CreateDirectory(String)

Creates a new directory (and any directories leading up to it). Note that when creating a directory somewhere under "Assets" in Unity Editor you should call AssetDatabase.Refresh() to make the directory show up - unless you create files in that directory and call AssetDatabase.Import() on them, in that case Refresh is not necessary.
